<h3 id="1-식별자와-데이터-타입">1. 식별자와 데이터 타입</h3>
<p>자바스크립트의 식별자 명명 규칙은 자바와 기본적으로 유사하다. 하지만 데이터 타입의 종류와 논리 평가 방식에서는 차이를 보인다.</p>
<ul>
<li><p><strong>데이터 타입의 분류</strong></p>
<ul>
<li><strong>기본 데이터형</strong>: <code>number</code>, <code>string</code>, <code>boolean</code>, <code>undefined</code>, <code>null</code>, <code>NaN</code></li>
<li><strong>참조 데이터형</strong>: 객체(배열, JSON), 함수, 클래스<blockquote>
<p>자바스크립트는 자바와 다르게 함수를 데이터로 취급한다. 즉, 함수 또한 변수에 저장이 가능하다. </p>
</blockquote>
</li>
</ul>
</li>
<li><p><strong>논리 평가 시스템 (Falsy 규칙)</strong>
논리 연산 시 참(<code>true</code>)과 거짓(<code>false</code>)이 아닌 일반 데이터도 논리값으로 취급한다.</p>
<ul>
<li><p><code>false</code>로 취급되는 값(<strong>Falsy</strong>): <code>0</code>, <code>&quot;&quot;</code>, <code>''</code>, <code>null</code>, <code>undefined</code>, <code>NaN</code> </p>
</li>
<li><p>위의 6가지 값을 제외한 모든 데이터는 <code>true</code> 로 평가된다.</p>
</li>
</ul>
</li>
</ul>

<h3 id="3-연산자와-자동-형변환">3. 연산자와 자동 형변환</h3>
<ul>
<li><strong>산술 연산 시 형변환</strong>
<code>+</code> 연산자는 문자열을 연결시키며, 나머지 연산자(<code>/</code>, <code>*</code>, <code>-</code>)는 수치 연산 과정에서 강제 형변환을 한다.</li>
</ul>

<ul>
<li><p><strong>단축 평가(Short-circuit evaluation)</strong>
논리 연산자(<code>&amp;&amp;</code>, <code>||</code>)가 단순히 <code>boolean</code>을 반환하지 않고, <strong>실제 데이터 값</strong>을 반환한다.</p>
<pre><code class="language-javascript">// &amp;&amp; (AND): 앞이 true면 뒤의 값을 반환한다.
let andResult = 10 &amp;&amp; &quot;홍길동&quot;;  // &quot;홍길동&quot;

// || (OR): 앞이 false면 뒤의 값을 반환한다. (기본값 설정 시 유용)
let orResult = NaN || 100;      // 100</code></pre>
</li>
</ul>

<h3 id="5-내장-객체의-활용-string-number-date">5. 내장 객체의 활용 (String, Number, Date)</h3>
<p>시스템 정의 객체는 자바의 클래스와 동일하게 속성과 메서드를 제공한다.</p>
<ul>
<li><p><strong>백틱 <code>`</code></strong>
복잡한 HTML 태그나 변수를 문자열로 표현할때, 가독성을 위해 백틱(<code>`</code>) 을 사용한다.</p>
<pre><code class="language-javascript">let name = &quot;홍길동&quot;; //[cite: 1]
let age = 20; //[cite: 1]

// + 연산자 대신 백틱과 ${}를 사용하여 직관적인 문자열 삽입이 가능하다.
let result = `이름:${name} 나이:${age}`; //[cite: 1]</code></pre>
</li>
</ul>
<h3 id="6-참조-데이터형-심화-array-json">6. 참조 데이터형 심화 (Array, JSON)</h3>
<ul>
<li><p><strong>배열(Array)과 가공 메서드</strong>
JS의 배열은 크기가 가변적이며 내부가 스택으로 관리된다(<code>push</code>, <code>pop</code>).</p>
<pre><code class="language-javascript">let scoreList = [14, 22, 65, 1, 34];

// 1. splice: 배열 요소의 삭제 및 삽입 (시작점, 삭제개수, 추가요소들...)
scoreList.splice(0, 3, 10, 20); // 0부터 시작해서 3개 삭제하고 10과 20 삽입[cite: 1]</code></pre>
</li>
<li><p><strong>JSON(JavaScript Object Notation) 표현식</strong>
<code>{key: value}</code> 형태를 가지며, value에는 배열, 함수, 클래스 등 모든 데이터 타입이 들어갈 수 있다. </p>
<pre><code class="language-javascript">let userInfo = { name: &quot;홍길동&quot;, age: 20 }; </code></pre>
</li>
</ul>
<h3 id="7-브라우저-객체-모델-bom">7. 브라우저 객체 모델 (BOM)</h3>
<p>BOM(Browser Object Model)은 브라우저가 열리면 자동으로 생성되는 브라우저 관련 객체 계층이다.</p>
<ul>
<li><p><strong>Window 객체</strong>
웹브라우저 계열 중 최상위 전역 객체(Global Object)다. </p>
<ul>
<li><p><strong>새 창 제어 (<code>window.open</code> / <code>window.close</code>)</strong>
<code>window.open(url, target, features)</code>를 통해 새 창을 열고, 반환된 객체를 변수에 저장해두면 부모 창에서 자식 창을 원격으로 제어하거나 닫을 수 있다.</p>
<pre><code class="language-javascript">let childWin; // 자식 창의 참조값을 저장할 전역 변수

function openChildWindow() {
    // url, 이름, 창의 크기(width/height) 및 위치(left/top)를 지정하여 새 창을 연다.
    childWin = window.open(&quot;target.html&quot;, &quot;&quot;, &quot;width=200, height=300, left=200, top=200&quot;); 
}

function closeSelfWindow() {
    // window.close()는 자기 자신 창을 닫는다.
    window.close(); 
}

function closeChildWindow() {
    // 저장해둔 참조 변수(childWin)를 통해 열어둔 자식 창을 닫는다.
    childWin.close();
}</code></pre>
</li>
<li><p><strong>타이머 제어</strong>: <code>setTimeout(function, delay)</code> (단 한번만 실행됨), <code>setInterval(function, delay)</code> (반복 실행)</p>
</li>
</ul>
</li>
<li><p><strong>기타 주요 하위 객체 (History, Location)</strong>
<code>screen</code>, <code>history</code>, <code>location</code>, <code>navigator</code>는 모두 Window 객체의 속성으로 제공되지만 전역객체이므로 생략이 가능하다.</p>
<ul>
<li><code>History</code>: 방문 기록 제어 (<code>history.go(-1)</code> : 뒤로 한번 이동 등)</li>
<li><code>Location</code>: 웹브라우저 URL 관련 객체로, 페이지 이동(라우팅)에 핵심적으로 사용된다.<pre><code class="language-javascript">// 버튼 클릭 이벤트 등에 연결하여 원하는 URL로 페이지를 이동시킨다.
function moveToPage() {
  // Location 객체의 href 속성에 URL을 재할당하면 해당 페이지로 이동한다.
  location.href = &quot;https://www.google.com&quot;;
}</code></pre>
</li>
</ul>
</li>
</ul>
